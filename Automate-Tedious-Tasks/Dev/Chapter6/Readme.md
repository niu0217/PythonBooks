## 2. 小程序-PigLatin

要求：

+ 如果单词以元音开头，就在单词末尾添加yay
+ 如果单词以辅音或者辅音簇（比如ch或gr）开头，那么该辅音或者辅音簇会移至单词的末尾，然后加上ay

举例：

```
输入：My name is AL SWEIGART and I am 4,000 years old.
输出：Ymay amenay isyay ALYAY EIGARTSWAY andyay Iyay amyay 4,000 yearsyay oldyay.
```
